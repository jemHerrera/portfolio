    About Sense of Neumor

    Sense of Neumor features Neumorphism, a design style associated with a realistic shadow effect. Neumorphism involves a shadow on one side of an element, as well as a highlight on the opposite side. For this to work the background color must be slightly darker than white to allow the highlight to become visible, creating a more realistic 3D effect.

    Dynamic Shadow

    To take this to the next level, we can allow the user to drag a light source element across the screen, which would result in a change of the angle of the shadow. The process involves computation of the coordinates and angle of the light source relative to the subject element, and then converting the values into css box-shadow syntax.

    Future Plans

    This project is still a work in progress. My plans for the next phase is to allow users to be able to adjust the light intensity, adjust the subject’s size and radius, as well as switch into dark mode. I also would love to include an auto-generated css box where users can copy the css for the given setup.
